construct_setup
===============
Setup construct core packages.

+---------------------+------------------------------+---------+
| package             | description                  | version |
+=====================+==============================+=========+
| construct_          | Core api                     | 0.1.17  |
+---------------------+------------------------------+---------+
| construct_cpenv_    | Cpenv Integration            | 0.1.1   |
+---------------------+------------------------------+---------+
| construct_launcher_ | Application launcher         | 0.1.5   |
+---------------------+------------------------------+---------+
| construct_maya_     | Autodesk Maya integration    | 0.1.5   |
+---------------------+------------------------------+---------+
| construct_nuke_     | The Foundry Nuke integration | 0.1.4   |
+---------------------+------------------------------+---------+
| construct_ui_       | Graphical user interface     | 0.1.5   |
+---------------------+------------------------------+---------+
| construct_ftrack_   | Ftrack integration           | N/A     |
+---------------------+------------------------------+---------+
| fsfs_               | File system metadata         | 0.2.2   |
+---------------------+------------------------------+---------+


Installation
============

Make sure git_ is installed and available from your command prompt. Then
from a terminal run:

.. code-block:: console

    pip install git+git://github.com/construct-org/construct_setup.git -I --process-dependency-links --trusted-host github.com
